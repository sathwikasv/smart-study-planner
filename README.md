**Smart Study Planner 📚**

**Project Description**
This is a web-based Smart Study Planner that helps students manage tasks, schedules, and deadlines efficiently.

**🚀 Features**
Add and manage tasks
Weekly planning
Study timer
Search functionality
🛠️ Technologies Used
HTML, CSS, JavaScript
Python (Flask)
MySQL
**▶️ How to Run**
Install requirements:
pip install -r requirements.txt
Run the application:
python app.py

**⚠️ Note**
.env file is not included for security reasons.
